# 📊 광고 정산 자동화 시스템

## 실행 방법

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 서버 실행
```bash
python app.py
```

### 3. 브라우저 접속
```
http://localhost:5000
```

---

## 사용 방법

### ① 회원가입 / 로그인
- 담당자마다 각자 계정을 만들면 데이터가 분리 저장됩니다

### ② 광고주 관리
- 광고주 추가 → 캠페인 추가 → 매체별 수수료율 입력
- 마크업 %: 광고주에게 청구하는 수수료율
- 매체수수료 %: 매체로부터 받는 수수료율 (15% 등)
- 페이백 %: 매체 페이백율 (5% 등)

### ③ 정산 처리
1. 광고주 / 캠페인 / 기간 선택
2. 각 매체별 공급가액 입력 (네이버 XLS 파일 파싱 지원)
3. 자동 계산 결과 확인
4. **XLSX 다운로드** → 세일즈양식 + 세발요청 시트 자동 완성

## 파일 구조
```
settlement_app/
├── app.py              # Flask 백엔드
├── requirements.txt    # 패키지 목록
├── settlement.db       # SQLite DB (자동 생성)
└── static/
    └── index.html      # 프론트엔드
```

## 지원 파일 형식
- 네이버 위임발행 XLS/XLSX → 자동 파싱 (계정명별 공급가액 집계)
- 구글 PDF → 공급가액 직접 입력
- 기타 매체 → 공급가액 직접 입력

## 출력 XLSX 시트
- **세일즈양식**: 기존 정산 양식과 동일한 컬럼 구조
- **세발요청**: 세금계산서 발행 요청 양식 자동 완성
